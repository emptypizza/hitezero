## 1. 프레임 단위 타임라인 분석

4fps 컨택트 시트로 전체 흐름을, 핵심 구간은 네이티브 ~30fps(67ms 스텝) 버스트로 분석.

| 구간 | 이벤트 |
|---|---|
| 0.0–1.0s | 전투 중. 플록이 자동 사격(머즐 글로우 점멸), 초록 점선 조준/이동선, 적은 빨간 디스크형 비행체. HUD: 해골 pill `0/1`, 플록 카운터 25, 좌측 엣지 자원 2종(35/38) |
| 1.0–3.2s | **레벨업 팝업** (게임 일시정지). 골드 배너 `レベルアップ!!` + 흰 라운드 카드 3장: `攻撃力↑` / `体力↑` / `攻撃力2倍 18秒`(임시 버프). 3.0s에 선택(파란 하이라이트) → 즉시 재개 |
| 3.2–9.8s | 엘리트(빨간 투구 괴물) 교전. 옐로 폭발 퍼프, 깃털 파편, 지면 스월 더스트 |
| **9.87s** | **처치 순간**: 해골 카운터 `0/1 → 1/1` 플립 + 시안 버블 팝 VFX + 보석(복셀 파편) 분출 |
| 10.1s | 우상단 토스트 슬라이드-인: **"Group Kill / Attack +90"** (그룹 전멸 보너스 스택) |
| 10.5–19.5s | 수집/휴지 페이즈. 보석 자석 수집, 자원 35/38 → 60/88, 플록 25 → 27로 성장 |
| 20.0s– | (영상 스크럽, 다른 장면) 플록 17마리 + 리더(파란 모자 오리), 점선 이동선, 파스텔 블록 지형 |
| 24.0–25.5s | 레벨업 2회차: `仲間の移動速度↑`(동료 이속) 등 3택 → 선택 즉시 재개 |
| 25.9–27.0s | 전투 버스트: 병아리별 머즐 글로우 67–134ms 주기 점멸, 시안 대형 버블 투척 스킬, 지면 폭발 링 |
| 27–34s | 교전 지속. `Group Kill Attack +90` 토스트 재등장, 자원 84/62 |

### 프레임 정밀 측정 (67ms 스텝 버스트)

| 항목 | 측정값 | 시사점 |
|---|---|---|
| 머즐 플래시 수명 | **1–2프레임 (33–134ms)** | 짧고 강한 발광 스프라이트 + 블룸. 길게 끌지 않아 연사감이 또렷 |
| 플록 전체 발광 케이던스 | 초당 8–15회 (매 프레임 어딘가 점멸) | "화면이 살아있다"는 인상의 원천. 개체별 발사 타이밍 디싱크 |
| 처치 체인 | 버블 팝(0ms) → 보석 분출(+67ms) → 카운터 플립(+67–134ms) → 토스트(+200–270ms) → 수집 시작(+400ms~) | 모든 처치가 ~0.3초 안에 3겹 보상 피드백으로 연결됨 |
| 레벨업 해제 | 선택 → **1–2프레임 내 전투 재개** | 전환 애니메이션 없음. 즉답성이 템포를 살림 |
| 토스트 | 슬라이드-인 ≤270ms, 수 초 유지 후 자동 소멸 | 전투를 가리지 않는 우상단 고정 위치 |

## 2. 스타일 분석 → 디자인 토큰

| 토큰 | 관찰값 |
|---|---|
| 배경 | 딥 네이비 나이트(#1A2230 계열), 습윤 반사 지면, 강한 비네트 |
| 보조색 | 파스텔 복셀 소품 — 핑크/연두/웜그레이 블록 (저채도, 비발광) |
| 발광색 | 웜 옐로(사격·폭발) + 시안(버블·보석) **2색으로 절제**. 적은 레드 |
| 셰이프 | 라운드 복셀, 아웃라인 없음, 소프트 섀도 |
| 앰비언스 | 시안/핑크 부유 입자(반딧불), 지면 스월 더스트 |
| 카메라 | 포트레이트 탑다운 ~60°, 타이트 팔로우 |
| UI | 라운드 필/카드, 흰 배경 + 볼드 다크 텍스트, 골드 배너(레벨업), 별 배지, 초록 ↑ 화살표, 우상단 토스트. 어두운 월드 위에서 흰 UI가 또렷 |
| QoL | 하단 중앙 `⏩ x2` 배속 토글, 하단 우측 음소거 — 인게임 상시 노출 |

핵심 문법: **"어두운 무대 + 절제된 발광 2색 + 파스텔 소품 + 흰 라운드 UI"**.
hitezero의 네온 다크(#0A0A16 + 시안/마젠타)와 같은 계열이라 이식 마찰이 적다.

## 3. 게임성 분석 — 시스템 분해

1. **플록 = 화력의 가시화**: 아군 수(17→27)가 곧 DPS이며 HUD 카운터로 성장이 보임
2. **오토 전투 + 조작 단순화**: 플레이어는 이동만, 사격은 자동. 초록 점선이 타깃/경로 피드백
3. **스테이지 목표 pill**: 상단 중앙 해골 `0/1` — 엘리트 처치가 명시적 클리어 조건
4. **레벨업 3택 로그라이트**: 영구 스탯(`攻撃力↑`, `体力↑`, `仲間の移動速度↑`)과 **임시 버프**(`攻撃力2倍 18秒`)를 섞어 제시
5. **자원 2종 누적**: 좌측 엣지 카운터, 처치/수집으로 증가 (메타 재화로 추정)
6. **Group Kill 보너스**: 그룹 전멸 시 `Attack +90` 스택 + 토스트 — 버스트 플레이에 보상
7. **모든 처치 = 루팅 모먼트**: 보석 분출 → 자석 수집의 짧은 쾌감 루프가 상시 반복
8. **QoL**: x2 배속/음소거 인게임 버튼 (모바일 세션 템포 존중)

## 4. Goal Plan — hitezero 이식

원칙: 장르 전환이 아니라 **DNA 이식**. 참조의 (a) 스타일 문법, (b) 게임필 타이밍 규격,
(c) 런 내 로그라이트 시스템을 hitezero(네온 나이프 브레이크아웃)에 단계 적용한다.
기존 자산(히트스톱·trauma² 셰이크·콤보·아이템 6종·메타 업그레이드 8종)은 유지하고 그 위에 쌓는다.

### Phase 0 — 스타일·타이밍 토큰 정립 (준비)
- **G0.1** `game_constants.gd`에 토큰 섹션 추가: 발광 2색 원칙(시안 주역 + 골드 보상),
  파스텔 보조색, VFX 타이밍 상수(`FLASH_MS≈67–134`, `KILL_CHAIN_BUDGET_MS=300`, `TOAST_IN_MS≤270`)
- AC: 이후 신규 VFX/UI가 토큰 상수만 참조 (하드코딩 0건)

### Phase 1 — 게임필 규격 이식 (최우선, ROI 최대)
- **G1.1 타격 플래시 규격화**: 머즐/히트 플래시를 1–2프레임(33–134ms) 수명으로 검증·조정
  - AC: 60fps 녹화 프레임 스텝에서 플래시 수명 ≤8프레임(웹 60fps 기준 134ms)
- **G1.2 처치 체인 규격화**: 블록 파괴 시 `팝(스케일 펀치) → 파편·코인 분출(+1프레임) → 점수/카운터 반응(+2프레임) → 자석 수집` 순차 체인
  - AC: 파괴 → 마지막 피드백 시작까지 ≤300ms, 모든 파괴에 루팅 모먼트(파편≥4개) 존재
- **G1.3 토스트 시스템 신설**: 우상단 슬라이드 카드 — 콤보 마일스톤/그룹킬/아이템 획득 통지
  - AC: 표시 ≤270ms, 자동 소멸, 다중 발생 시 수직 스택
- **G1.4 앰비언스 강화**: 비네트 + 부유 입자(기존 배경 별 입자 확장), 발광 2색 절제 점검
  - AC: 정지 화면에서도 입자 2종 이상 미세 운동, 동시 발광 색 ≤2

### Phase 2 — HUD/UX 어댑션
- **G2.1 목표 pill**: 상단 중앙 `★ 0/3`(잔여 별) pill + 달성 시 플립 애니메이션
  - AC: 별 수집 순간 pill이 1프레임 내 반응(플립/펀치)
- **G2.2 x2 배속 + 음소거 버튼**: 하단 상시 버튼, SHOOTING 페이즈 한정 `Engine.time_scale=2`
  - AC: 배속 중에도 히트스톱·셰이크 체감 유지(타임스케일 독립 처리), 토글 상태 시각 표기
- **G2.3 UI 라운드 카드 통일**: 오버레이(스테이지 클리어/게임오버/레벨업)를 흰 라운드 카드 + 볼드 텍스트 문법으로 통일. Press Start 2P는 헤드라인 전용 유지

### Phase 3 — 런 내 로그라이트 (게임성 핵심 이식)
- **G3.1 레벨업 3택**: 스테이지 클리어 시 일시정지 + 3택 카드.
  영구(런 한정): 나이프+1 / 속도+10% / 트레이+10 ··· 임시: 공격력 2배 18초 / 관통 12초
  - AC: 선택 → 1–2프레임 내 게임 재개(전환 연출 없음), 런 모디파이어는 `Session` 메타와 분리된 구조
- **G3.2 그룹킬 보너스**: 한 발사 사이클(또는 0.5s 윈도) 내 N개 동시 파괴 시 런 내 공격력 스택 + 토스트
  - AC: 판정 윈도 상수화, 스택 수 HUD 표기, 토스트 G1.3 재사용
- **G3.3 임시 버프 타이머 HUD**: 기존 아이템 6종 구조 재사용해 잔여 시간 표기

### Phase 4 — 스타일 심화 (선택)
- **G4.1** 지면 반사/습윤 그라데이션 (저비용 페이크 반사)
- **G4.2** 블록 라운드 코너 + 소프트 섀도 — 네온 정체성 유지 범위 내
- **G4.3** 엘리트/보스 전용 처치 버블 팝(시안 링 + 버블) VFX

### Phase 5 — 검증
- 웹 빌드 60fps 유지(x2 배속 포함), 모바일 터치 QA(토스트/3택 카드 터치 영역 ≥44px),
  전 Phase AC 체크리스트 일괄 통과, `arcade_feel_polish_plan.md`와 정합 확인

### (분기 옵션) Track F — 덕 플록 프로토타입
참조의 게임성 자체(플록 오토슈터)를 검증하고 싶다면 **별도 신규 프로토**로:
리더 드래그 이동 + 팔로워 N마리 오토샷 + XP 보석 → 3택 + 엘리트 목표 pill.
동일 디자인 토큰 재사용, 1주 스파이크 범위. hitezero 본편과는 독립 의사결정.
