# HiteZero — 폴리싱 작업리스트 v1.1+ (출시 이후 성장 10)

> 2026-06-15. v1.0 출시 폴리싱(P1~P10)은 구현/스펙 완료. 이건 **그 다음 10** — 출시 후 리텐션·콘텐츠·도달을 올리는 성장 작업.
> 코드 직독 기반(실제 시스템에 매핑): `session.gd`(save·UPGRADES·stats·coins) · `boss.gd`(26KB) · `level_generator.gd` · `game_root.gd` · `hud.gd` · `title.gd`.

## Tier 1 — v1.1 (리텐션, 다음 스프린트 최우선)

- **근거**: 온라인 리더보드 불가(완전 오프라인). 리텐션 훅이 "최고점 갱신"뿐 → 매일 돌아올 이유 부족.

- 임팩트 高 / 노력 中 / 담당 나

### V2. 게임오버 런 요약 + 점수 공유 카드 (P9 구현)
- **근거**: 게임오버가 즉시 타이틀로(밋밋). 공유 메커니즘 0.
- **무엇**: 게임오버 시 요약(스테이지·점수·콤보·최고갱신 연출) + **공유용 PNG 카드 생성**(점수+보스 컷) → OS 공유 시트. 오프라인 바이럴.
- 임팩트 中~高(D1/D7·신규유입) / 노력 中 / 담당 나

### V3. 업적 시스템 (로컬, stats 기반)
- **근거**: `session.gd`에 이미 stats 6종(runs·blocks·enemies·bosses·combo·items) 적재 중 — **업적의 데이터가 이미 있음**.
- **무엇**: 마일스톤 업적(예 블록 1k·보스 10·콤보 x10) + 달성 시 코인 보상 → 코인경제와 결합해 단기 목표 제공. 기존 Stats 모달 확장.

- **근거**: 셰이크 토글이 지금 How-To 모달에 묻혀있음(P6). 사운드 뮤트는 HUD. 분산.
